# Image Prompt Lab v0.1.0

## Highlights

- First macOS desktop release for the local-first Image Prompt Lab Gallery MVP.
- Packages the Tauri desktop app through GitHub Releases with updater metadata.
- Establishes the managed local resource library model for generated and imported image assets.

## User-facing changes

- Added the Studio Console desktop shell for Gallery, Albums, Review, Generate, Queue, and Settings workflows.
- Added local resource libraries backed by SQLite and managed filesystem storage.
- Added Gallery browsing, search, metadata review, album workflows, image lightbox inspection, and masonry-style image browsing.
- Added text-to-image generation through the fake provider and the Codex CLI provider boundary.
- Added image-to-image versioning, uploaded reference handling, and asset version lineage.
- Added Settings library management for opening, registering, repairing, and inspecting local libraries.
- Added a CLI for library initialization, import, search, generation, album, and metadata workflows.

## Fixes

- Fixed Gallery search preview filtering so filtered previews match the current query.
- Fixed desktop window chrome to use native window controls.
- Hardened Album and Review workflows around staged metadata review and library mutation behavior.
- Improved desktop workbench layout so core workflows remain usable on narrower laptop viewports.

## Developer and release notes

- Added OpenSpec-driven product and architecture workflow documentation.
- Added the initial macOS release automation, updater artifacts, and release verification path.
- Added core architecture and performance cleanup work that shaped the current DDD boundaries.
