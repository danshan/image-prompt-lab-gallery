# Image Prompt Lab v0.1.6

## Highlights

- Adds Prompt Workspace as a first-class prompt authoring and lineage surface.
- Adds scheduled image generation automation through the daemon task runtime.
- Redesigns the desktop interface system for a denser, bilingual Studio Console.
- Adds library content lifecycle operations for safer library import, merge, deduplication, and cleanup workflows.

## User-facing changes

- Added prompt documents, prompt versions, prompt variables, and prompt-to-generation lineage.
- Added Prompt Workspace UI actions for creating, versioning, running, and saving prompt work.
- Exposed prompt lineage in asset detail so generated images can point back to the prompt version that produced them.
- Added scheduled image generation with schedule state, queue integration, daemon execution, and next-run behavior.
- Added a redesigned desktop shell with command bar, context drawer, compact navigation, light/dark themes, and English/Chinese locale support.
- Added resource library content lifecycle workflows for comparing library contents, merging assets, resolving duplicates, and managing imported content.
- Updated the Codex provider generation parameters used by generation tasks.

## Fixes

- Fixed prompt persistence boundaries and prompt variable wrapper schema compatibility.
- Fixed desktop task drafts so prompt versions are preserved during generation.
- Fixed Prompt Workspace run flow failure handling.
- Fixed prompt snapshot event IDs so prompt lineage remains stable.
- Fixed Prompt Workspace layout issues on laptop-size viewports.
- Fixed daemon scheduler coverage so scheduled generation uses real prompt versions.

## Developer and release notes

- Added prompt domain models, policies, use cases, SQLite persistence, Tauri commands, and desktop workflow state.
- Added scheduled generation domain models, policies, use cases, SQLite persistence, daemon runtime integration, and scheduler tests.
- Added library merge and content lifecycle core modules with persistence and CLI integration.
- Archived the Prompt Workspace, scheduled generation, desktop redesign, layout fix, and library lifecycle OpenSpec changes.
- Fast-forwarded `main` to this release commit after the release branch had already been tagged and published.

## Install

Download the attached macOS asset from this release. Current builds use macOS ad-hoc signing plus Tauri updater signing, not Apple Developer ID notarization, so the first launch may require the standard macOS Gatekeeper override.
