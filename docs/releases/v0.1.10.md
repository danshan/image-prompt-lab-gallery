# Image Prompt Lab v0.1.10

## Highlights

- Replaces the top workspace switcher with a collapsible responsive sidebar for Gallery, Albums, Prompts, Schedules, Review, Queue, and Settings.
- Moves Generate into a lightweight modal with provider, prompt, and reference image inputs.
- Removes persistent overview counters from workspace content and keeps collection counts in compact sidebar labels.

## User-facing changes

- Sidebar supports expanded and collapsed modes, including icon-only navigation and theme/language actions.
- The top bar keeps library switching available without duplicating the app brand.
- Generate reference image thumbnails stay fixed-height so the modal remains compact.
- Chinese and English labels cover the new sidebar and Generate modal surfaces.

## Fixes

- Schedules form select controls now use the same styling as the rest of the desktop app.
- Reference images in Generate no longer expand the modal vertically.
- The temporary `/tmp/image-prompt` library path can be repaired or initialized without blocking the desktop preview flow.

## Developer and release notes

- Archived the `sidebar-generate-modal` OpenSpec change.
- Synced the desktop workbench spec for Responsive Sidebar, Generate modal, and overview counter placement.
- Updated desktop frontend tests for responsive shell helpers and i18n coverage.

## Install

Download the attached macOS asset from this release. Current builds use macOS ad-hoc signing plus Tauri updater signing, not Apple Developer ID notarization, so the first launch may require the standard macOS Gatekeeper override.
