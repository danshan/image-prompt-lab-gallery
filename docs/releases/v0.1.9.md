# Image Prompt Lab v0.1.9

## Highlights

- Adds Settings / Task Queue concurrency controls for daemon task execution.
- Allows the daemon scheduler to claim and execute multiple eligible tasks concurrently while preserving provider-level safety limits.
- Fixes scheduled generation output post-processing so run-now and background reconciliation apply album membership and canonical tags consistently.

## User-facing changes

- Settings now includes a Task Queue section for max parallel task configuration.
- Review metadata detail remains reachable through internal scrolling on compact desktop viewports.
- Schedules job rows are easier to scan, and editing an existing schedule has an explicit New schedule path.

## Fixes

- Task Queue settings now show a recoverable offline/runtime-required state in browser preview instead of staying in Loading.
- Scheduled generation run outputs are reconciled idempotently, including album membership, tags, output rows, and counters.
- Review metadata detail actions and suggestion history remain reachable at the 960px desktop target.

## Developer and release notes

- Archived the `configure-task-queue-concurrency`, `fix-review-metadata-scroll`, `fix-scheduled-output-post-processing`, and `fix-schedules-job-editor-ux` OpenSpec changes.
- Synced the affected main specs for desktop workbench, metadata review, scheduled image generation, and task manager daemon.
- Added daemon and core tests for task queue settings, scheduler concurrency, atomic task claiming, and scheduled output reconciliation.

## Install

Download the attached macOS asset from this release. Current builds use macOS ad-hoc signing plus Tauri updater signing, not Apple Developer ID notarization, so the first launch may require the standard macOS Gatekeeper override.
